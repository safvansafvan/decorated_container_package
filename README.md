# Decorated Container

A Flutter utility library for creating and managing decorated containers.

This package provides a set of functions and widgets that make it easy to create containers with custom decorations, such as borders, gradients, and shadows. It's designed to simplify container styling in Flutter applications.

## Features

- Easily create containers with custom decorations.
- Support for borders, gradients, and shadows.
- Highly customizable and flexible.

  decorated_container: ^0.0.2
